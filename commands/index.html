This page allows you to create shareable links that can open and run commands in Lumafly

For this to work you will need to download Lumafly from download page and set it up

The commands you can use with Lumafly:

General Commands:

Open Lumafly and import a modpack.

Open Lumafly and download the mods provided.

Support Commands:

Opens Lumafly and reinstalls all mods. Could help fix issues that happened because mods are not downloaded correctly.

Opens Lumafly and reset its persistent settings. Equivalent to downloading and opening Scarab for the first time.

Other Commands:

You can use this link to redirect to any scarab command
